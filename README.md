# Vilinq Prototype Demo

This repository contains a **single-file, browser-ready demo** of the Vilinq childcare payment flexibility prototype.

## Quickest way to view

No local server is required.

1. Download this repo (or just `vilinq-demo.html`).
2. Double-click `vilinq-demo.html` to open it in your default browser.

You can also open `index.html` directly the same way.

## Shareable demo file

- `vilinq-demo.html` → standalone file intended for sharing via email, Slack, Drive, etc.
- `index.html` → same prototype used as the primary source in this repo.

## What’s included in the demo

- Vilinq hero section and branding
- Provider Dashboard view
- Parent Payment View
- Why Now and How Vilinq Works sections
- Clickable interactions for:
  - switching between Provider and Parent views
  - selecting payment options

## Notes

- This is a prototype with sample data only.
- No real payment processing is connected.
